## 해결 과정
 - mapIndexed, sum을 이용해 계산

## 코드 1

```kotlin
fun solution(a: IntArray, b: IntArray): Int {
    var answer: Int = 1234567890

    answer = a.mapIndexed { idx, aNum ->
        aNum * b[idx]
    }.sum()

    return answer
}
```

## 코드 2: 다른 사람의 코드

```kotlin
fun solution(a: IntArray, b: IntArray): Int {
    return a.zip(b).map { it.first * it.second }.sum()
}
```

## 배운 점
- zip을 이용하여 계산 가능하다.
